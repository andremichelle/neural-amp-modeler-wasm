# Input Audio Files

This directory contains sample input audio files for demonstrating the Neural Amp Modeler WASM.

## Contributing Input Files

We welcome contributions of high-quality input audio files! These samples help showcase the capabilities of the amp modeler and provide users with diverse testing material.

### File Specifications

Please ensure your contributions meet the following specifications:

- **Duration**: 20-30 seconds
- **Sample Rate**: 48 kHz
- **Bit Depth**: 24-bit
- **Channels**: Mono
- **Format**: WAV

### How to Contribute

1. Fork this repository
2. Add your audio file(s) to the `ui/public/inputs/` directory
3. Update this README with your contribution details (see Contributors section below)
4. Submit a pull request

### Ideal Content

We're looking for a variety of input samples including:

- Clean guitar riffs and licks
- Rhythm guitar parts
- Lead guitar phrases
- Bass guitar lines
- Different playing styles (fingerstyle, palm muting, strumming, etc.)
